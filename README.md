# Ubuntu Prism

Ubuntu Prism is an open-source project to transform Ubuntu into a modern, elegant desktop with:

- ✨ Glassmorphism
- 🎨 Beautiful gradients
- 🚀 Smooth animations
- 🪟 Floating panels
- 🔔 Modern notifications
- 🎯 Redesigned Quick Settings
- 🎭 New icon pack
- 🌈 Dynamic accent colors
